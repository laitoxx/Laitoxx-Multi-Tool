Laitoxx — это мощный инструмент для OSINT (разведки по открытым источникам) и кибербезопасности с современным графическим интерфейсом (GUI). Предназначен для образовательных и исследовательских целей, помогая собирать общедоступные данные для анализа безопасности и оценки цифрового следа.

⚠ **Дисклеймер**: Инструмент создан исключительно в образовательных целях. Разработчики не несут ответственности за его использование.

### 🔌 Разработка плагинов (Lua)
Laitoxx поддерживает расширение функционала через систему плагинов на Lua. 
Руководство: 👉 [Руководство по плагинам (Русский)](pluginBuilding.ru.md)

### 🔹 Функционал
* **Современный GUI**
* **Расширяемость через Lua-плагины**
* **Настраиваемые темы**
* **Редактор графов (Визуализация связей)**
* **Поддержка SOCKS5 прокси**
* **Режим производительности**

#### OSINT Инструменты:
* Глобальный поиск Wi-Fi и MAC-адресов
* Продвинутый просмотрщик метаданных и криминалистика
* Поиск по номеру телефона
* IP-трекинг и сканирование (с интерактивной картой)
* Email OSINT и валидация
* Telegram OSINT
* Поиск по нику и генерация никнеймов (по 500+ сайтам)
* Поиск по изображению
* Google OSINT (продвинутые дорки)
* Поиск по локальным базам данных
* Веб-краулер (Web-crawler)
* Информация о сайте (Whois и др.)

#### Веб и Сеть:
* Сканер портов (на базе Nmap)
* Nmap-сканер
* Поиск субдоменов
* Детектор CMS
* Определение технологий (WAF, CDN и др.)
* Инструменты веб-безопасности (SSL, CORS, Open Redirect, HTTP инспектор, JWT анализатор)
* Калькулятор CIDR
* REGEX Tester

#### Утилиты:
* Работа с хешами (Хеширование, Идентификация, Rainbow Table)
* Преобразователь текста (Leet, Азбука Морзе, Base64, Hex, ROT-13)
* Генератор паролей

### 🚀 Установка и Запуск

Laitoxx предоставляет удобные скрипты автоматической установки для всех основных операционных систем. Вы можете скачать инструмент через Git или загрузив ZIP-архив.

#### 🔽 Вариант 1: Через Git (Рекомендуется)
1. **Клонируйте репозиторий:**
   ```sh
   git clone https://github.com/Laitoxx/Laitoxx-Multi-Tool.git
   cd Laitoxx-Multi-Tool
   ```

#### 🔽 Вариант 2: Без Git
1. Скачайте репозиторий в виде ZIP-архива с GitHub (Нажмите **Code** -> **Download ZIP**).
2. Распакуйте архив и откройте терминал в появившейся папке (`Laitoxx-Multi-Tool-main`).

---

#### 🐧 Ubuntu/Debian (Linux)
Запустите скрипт автоматической установки:
```sh
chmod +x install.sh
./install.sh
```
*Для последующего запуска используйте: `python3 start.py`*

#### 🍎 MacOS
Запустите скрипт автоматической установки:
```sh
chmod +x install.sh
./install.sh
```
*Для последующего запуска используйте: `python3 start.py`*

#### 🪟 Windows
Дважды кликните по файлу `install.bat` или запустите его через командную строку:
```cmd
install.bat
```
*Для последующего запуска используйте: `python start.py`*

### ✨ Список изменений (Changelog) v2.3.1
* **Глобальный поиск Wi-Fi**: Отслеживайте Wi-Fi роутеры по всему миру. Теперь вы можете узнавать точные координаты сотен соседних роутеров, имея на руках лишь IP или один MAC-адрес.
* **Умные интерактивные карты**: Карты теперь автоматически подсвечивают жилые комплексы, кафе, парки и найденные точки Wi-Fi, чтобы упростить физическую визуализацию объектов.
* **Продвинутый просмотрщик метаданных**: Добавлен мощный инструмент для извлечения, анализа и безопасного удаления скрытых метаданных (Exif, GPS, авторы) из фото и документов.
* **Улучшения интерфейса**: Добавлены плавные анимации загрузки и фоновая обработка задач — приложение больше не зависает во время тяжелых поисковых запросов.
* **Мультиязычность**: Окна сетевой разведки полностью переведены на английский, русский, украинский и турецкий языки с динамическим переключением.
* **Улучшение производительности**: Масштабная внутренняя оптимизация кода для более быстрой и стабильной работы всей программы.
