Laitoxx — це потужний інструмент для OSINT (розвідки за відкритими джерелами) та кібербезпеки із сучасним графічним інтерфейсом (GUI). Призначений для освітніх та дослідницьких цілей, допомагає збирати загальнодоступні дані для аналізу безпеки та оцінки цифрового сліду.

⚠ **Відмова від відповідальності**: Інструмент створено виключно в освітніх цілях. Розробники не несуть відповідальності за його використання.

### 🔌 Розробка плагінів (Lua)
Laitoxx підтримує розширення функціоналу через систему плагінів на Lua. 
Керівництво: 👉 [Керівництво з плагінів (Українська)](pluginBuilding.uk.md)

### 🔹 Функціонал
* **Сучасний GUI**
* **Розширюваність через Lua-плагіни**
* **Налаштовувані теми**
* **Редактор графів (Візуалізація зв'язків)**
* **Підтримка SOCKS5 проксі**
* **Режим продуктивності**

#### OSINT Інструменти:
* Глобальний пошук Wi-Fi та MAC-адрес
* Просунутий переглядач метаданих
* Пошук за номером телефону
* IP-трекінг та сканування (з інтерактивною картою)
* Email OSINT та валідація
* Telegram OSINT
* Пошук за ніком та генерація нікнеймів (по 500+ сайтам)
* Пошук за зображенням
* Google OSINT (просунуті дорки)
* Пошук по локальних базах даних
* Веб-краулер (Web-crawler)
* Інформація про сайт (Whois тощо)

#### Веб та Мережа:
* Сканер портів (на базі Nmap)
* Nmap-сканер
* Пошук субдоменів
* Детектор CMS
* Визначення технологій (WAF, CDN тощо)
* Інструменти веб-безпеки (SSL, CORS, Open Redirect, HTTP інспектор, JWT аналізатор)
* Калькулятор CIDR
* REGEX Tester

#### Утиліти:
* Робота з хешами (Хешування, Ідентифікація, Rainbow Table)
* Перетворювач тексту (Leet, Азбука Морзе, Base64, Hex, ROT-13)
* Генератор паролів

### 🚀 Встановлення та Запуск
1. **Клонуйте репозиторій:**
   ```sh
   git clone https://github.com/Laitoxx/Laitoxx-Multi-Tool.git
   ```
2. **Перейдіть до директорії проєкту:**
   ```sh
   cd Laitoxx-Multi-Tool
   ```
3. **Встановіть залежності:**
   ```sh
   pip install -r requirements.txt
   ```
4. **Запустіть застосунок:**
   ```sh
   python start.py
   ```

### ✨ Список змін (Changelog) v2.3.1
* **Глобальний пошук Wi-Fi**: Відстежуйте Wi-Fi роутери по всьому світу. Тепер можна отримати точні координати сотень сусідніх роутеров, маючи лише IP або один MAC-адресу.
* **Розумні інтерактивні карти**: Карти тепер автоматично підсвічують житлові будинки, кафе, парки та знайдені точки Wi-Fi для зручної візуалізації об'єктів.
* **Просунутий переглядач метаданих**: Додано потужний інструмент для вилучення, аналізу та безпечного видалення прихованих метаданих (Exif, GPS, автори) із зображень та документів.
* **Покращення інтерфейсу**: Додані плавні анімації завантаження та фонова обробка запитів, що забезпечує стабільну роботу програми під час тривалих пошуків.
* **Багатомовність**: Вікна мережевої розвідки повністю перекладено англійською, російською, українською та турецькою мовами.
* **Покращення продуктивності**: Глобальна оптимізація та рефакторинг коду для швидшої і стабільнішої роботи.
