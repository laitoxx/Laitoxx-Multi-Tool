## 1. Observation
The following details were observed from the codebase:
- **Graph Editor Structure**: `GraphEditorWindow` (`src/laitoxx/interfaces/gui/graph_editor.py:1232`) is a `QDialog` that sets up a left panel, center panel (holding `MermaidView`), and right panel. It does not currently accept drops or define drag-and-drop handlers.
- **Web Canvas & Bridge**: `MermaidView` (`src/laitoxx/interfaces/gui/graph_editor.py:958`) contains a `QWebEngineView` component (`self._web`) which displays the HTML page generated by `generate_html(...)`. `tests/test_web_bridge.py` shows that the bridge propagates user interaction (node/edge selection) via `QWebChannel`.
- **Vis-Network Setup**: `generate_html` (`src/laitoxx/shared/graph/mermaid.py:101`) constructs the vis-network graph, mapping node styles and shapes. It exposes `window.selectNode` and `window.selectEdge` (lines 481-486) but does not provide dynamic insertion methods.
- **Metadata Viewer Export**: `_export_to_graph` in `src/laitoxx/features/utilities/metadata_viewer/gui_window.py:283` is the baseline logic for converting metadata results into the python graph model:
  - `file_node` of type `"Document"` (`line 330`).
  - Linked `Person` nodes for `"Author"`, `"Creator"`, `"Producer"`, `"OwnerName"` (`line 341`).
  - Linked `Custom` nodes with `mermaid_shape="hexagon"` for `"EXIF:Software"`, `"Software"`, `"Tika:creator"`, `"Hachoir:Software"` (`lines 354-358`).
- **Metadata Engine**: `MetadataEngine` (`src/laitoxx/features/utilities/metadata_viewer/engine.py:86`) and its default instance `engine_instance` provide synchronous extraction of metadata dictionaries.

---

## 2. Logic Chain
1. **Drag-and-Drop Interception**: To enable file drops anywhere on the Graph Editor, we configure `setAcceptDrops(True)` on `GraphEditorWindow`. Since child widgets (like the `QWebEngineView` inside `MermaidView` or `QTextEdit`) consume or block drops by default, we install an event filter on `self._mermaid_view._web` and `self._mermaid_view._text` in `GraphEditorWindow._build_ui` to catch `DragEnter`, `DragMove`, and `Drop` events first, forwarding them directly to `GraphEditorWindow`'s event handlers.
2. **Supported File Validation**: Within `dropEvent(self, event)`, we retrieve URL files. If any file has an extension not belonging to the supported list (`.pdf`, `.png`, `.jpg`, `.jpeg`, `.tiff`, `.mp3`, `.wav`, `.mp4`, `.avi`, `.doc`, `.xls`, `.ppt`, `.docx`, `.xlsx`, `.pptx`, `.exe`, `.dll`, `.sys`), we instantly trigger a `QMessageBox.warning` and call `event.ignore()` / abort the operation.
3. **Graph Model Construction**: For supported files, we invoke `engine_instance.extract_metadata(path)` and build:
  - A `"Document"` node whose label is the file name, storing all other metadata fields inside `node.metadata` to ensure they are visible in property sheets/tooltips.
  - Linked `"Person"` nodes (created/edited relation) for author keys.
  - Linked `"Custom"` nodes with `mermaid_shape="hexagon"` (created with relation) for software keys.
  - Deduping is handled by checking existing node labels/types in the `Graph` object.
4. **Dynamic Canvas Update**: Instead of calling `_refresh_all()` which triggers `setHtml()` and completely reloads the vis-network iframe (wiping out zoom/physics states), we introduce window-level JS helpers: `window.addNode(n)` and `window.addEdge(e)` inside the HTML generation template in `mermaid.py`. Python wrappers in `MermaidView` will invoke these helpers via `self._web.page().runJavaScript(js_code)` dynamically.
5. **Headless Testing**: We design a new pytest test module `tests/test_drag_drop_import.py` using `QDropEvent` and mock `QMimeData` containing local file URLs. We patch `QMessageBox.warning` and `MetadataEngine.extract_metadata` to verify successful imports, warning alerts, and graph state updates safely under the offscreen QPA platform.

---

## 3. Caveats
- **Async vs Sync Processing**: Metadata extraction is synchronous. If a user drops a massive batch of files (e.g. hundreds of files), it could temporarily freeze the GUI thread. A future enhancement could run extraction in a background `QThread`.
- **Headless Limits**: Testing QWebEngineView JS interactions requires an active Qt event loop (`QEventLoop`), which is fully supported under the `offscreen` QPA platform.

---

## 4. Conclusion
The proposed architecture provides seamless drag-and-drop metadata extraction, ensures clean graph models, preserves user view state via dynamic Vis-Network updates, and remains fully testable in headless CI environments.

```python
    def eventFilter(self, watched, event) -> bool:
        from PyQt6.QtCore import QEvent
        if watched in (getattr(self._mermaid_view, "_web", None), getattr(self._mermaid_view, "_text", None)):
            if event.type() == QEvent.Type.DragEnter:
                self.dragEnterEvent(event)
                return True
            elif event.type() == QEvent.Type.DragMove:
                self.dragMoveEvent(event)
                return True
            elif event.type() == QEvent.Type.Drop:
                self.dropEvent(event)
                return True
        return super().eventFilter(watched, event)

    def dragEnterEvent(self, event):
        if event.mimeData().hasUrls():
            event.acceptProposedAction()
        else:
            super().dragEnterEvent(event)

    def dragMoveEvent(self, event):
        if event.mimeData().hasUrls():
            event.acceptProposedAction()
        else:
            super().dragMoveEvent(event)

    def dropEvent(self, event):
        if event.mimeData().hasUrls():
            urls = event.mimeData().urls()
            valid_files = []
            supported_extensions = {
                ".pdf", ".png", ".jpg", ".jpeg", ".tiff", 
                ".mp3", ".wav", ".mp4", ".avi", 
                ".doc", ".xls", ".ppt", ".docx", ".xlsx", ".pptx", 
                ".exe", ".dll", ".sys"
            }
            for url in urls:
                filepath = url.toLocalFile()
                if filepath:
                    _, ext = os.path.splitext(filepath)
                    ext = ext.lower()
                    if ext not in supported_extensions:
                        QMessageBox.warning(
                            self,
                            _t("ge_unsupported_file_title", default="Unsupported File Type"),
                            _t("ge_unsupported_file_msg", default="File type '{ext}' is not supported.").format(ext=ext)
                        )
                        event.ignore()
                        return
                    valid_files.append(filepath)
            
            if valid_files:
                event.acceptProposedAction()
                self._import_dropped_files(valid_files)
        else:
            super().dropEvent(event)

    def _import_dropped_files(self, filepaths: list[str]):
        from laitoxx.features.utilities.metadata_viewer.engine import engine_instance
        from laitoxx.shared.graph.model import Node, Edge
        import uuid

        existing_nodes = {n.label: n for n in self._graph.nodes}
        added_nodes = 0
        added_edges = 0
        
        new_nodes = []
        new_edges = []

        for filepath in filepaths:
            data = engine_instance.extract_metadata(filepath)
            if "error" in data:
                continue

            # File node
            file_node_id = f"Doc_{uuid.uuid4().hex[:8].upper()}"
            file_node = Node(
                id=file_node_id,
                label=data.get("FileName", os.path.basename(filepath)),
                node_type="Document"
            )
            file_node.metadata = {
                "FilePath": data.get("FilePath", filepath),
                "FileSize": str(data.get("FileSize", 0)),
                "MIMEType": data.get("MIMEType", ""),
                "MD5": data.get("MD5", "")
            }
            # Add extra metadata fields
            for k, v in data.items():
                if k not in ["FilePath", "FileName", "FileSize", "MIMEType", "MD5", "ExtractedWith"]:
                    file_node.metadata[k] = str(v)

            self._graph.add_node(file_node)
            new_nodes.append(file_node)
            added_nodes += 1

            # Extract Authors
            for key in ["Author", "Creator", "Producer", "OwnerName"]:
                val = data.get(key)
                if val:
                    val_str = str(val).strip()
                    if not val_str:
                        continue
                    if val_str in existing_nodes and existing_nodes[val_str].node_type == "Person":
                        author_node = existing_nodes[val_str]
                    else:
                        author_id = f"Pers_{uuid.uuid4().hex[:8].upper()}"
                        author_node = Node(id=author_id, label=val_str, node_type="Person")
                        self._graph.add_node(author_node)
                        new_nodes.append(author_node)
                        existing_nodes[val_str] = author_node
                        added_nodes += 1

                    edge_id = f"Edge_{uuid.uuid4().hex[:8].upper()}"
                    edge = Edge(id=edge_id, source_id=author_node.id, target_id=file_node.id, label="created/edited", edge_type="Connected")
                    self._graph.add_edge(edge)
                    new_edges.append(edge)
                    added_edges += 1

            # Extract Software
            for key in ["EXIF:Software", "Software", "Tika:creator", "Hachoir:Software"]:
                val = data.get(key)
                if val:
                    val_str = str(val).strip()
                    if not val_str:
                        continue
                    if val_str in existing_nodes and existing_nodes[val_str].node_type == "Custom":
                        software_node = existing_nodes[val_str]
                    else:
                        software_id = f"Soft_{uuid.uuid4().hex[:8].upper()}"
                        software_node = Node(id=software_id, label=val_str, node_type="Custom", mermaid_shape="hexagon")
                        self._graph.add_node(software_node)
                        new_nodes.append(software_node)
                        existing_nodes[val_str] = software_node
                        added_nodes += 1

                    edge_id = f"Edge_{uuid.uuid4().hex[:8].upper()}"
                    edge = Edge(id=edge_id, source_id=file_node.id, target_id=software_node.id, label="created with", edge_type="Connected")
                    self._graph.add_edge(edge)
                    new_edges.append(edge)
                    added_edges += 1

        # Refresh lists & text raw view
        self._refresh_nodes_list()
        self._refresh_edges_list()
        self._raw_code.setPlainText(generate_mermaid(self._graph))

        # Push to canvas dynamically via QWebChannel
        accent = self._theme.get("accent_color", _ACCENT)
        for node in new_nodes:
            self._mermaid_view.add_node_js(node)
        for edge in new_edges:
            self._mermaid_view.add_edge_js(edge, accent)

        self._set_status(f"Imported {added_nodes} nodes and {added_edges} edges from dropped files.")
```
